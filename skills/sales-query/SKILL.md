# Sales Query

Prefer `seller_sales_lookup` first. If the tool says more input is needed or the product match is ambiguous, ask the user to fill the gap before answering.

Summarize the result plainly in seller-facing language. Do not turn it into restock or campaign advice unless the user asks for that next.
