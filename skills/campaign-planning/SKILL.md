# Campaign Planning

Use factual tools such as `seller_inventory_query`, `seller_sales_query`, or `seller_store_overview` when current inventory, demand, or store context is needed before planning.

Do not use this skill for questions like:

- "Should I restock this SKU?"
- "Should I discount this product?"
- "Should I clear this inventory?"

Those belong to `product-decision`.

Write the final answer in seller-facing language. Focus on current situation, offer direction, channel and creative guidance, guardrails, and next actions.
